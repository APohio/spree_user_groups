SpreeUserGroups
===============

Introduces user groups and calculator to add adjustment for specified user group.


Installation
============

      Add `gem "spree_user_groups", '>=0.40.1'`
      Run `bundle install`
      Run `rake spree_user_groups:install`
      Run `rake db:migrate`
